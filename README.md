# Product-list-with-cart
A responsive e-commerce frontend where users can browse products, add items to a shopping cart, manage quantities, view total price, and remove items. Built with HTML, SCSS, and Vanilla JavaScript for a clean, interactive UI.
## Features
- Display a list of products with images, names, and prices
- Add products to the shopping cart with quantity management
- View cart summary including total price
- Remove items from the cart
- Persist selected items using Local Storage, keeping the cart state after page reloads
- Responsive design for desktop and mobile devices
- Smooth and user-friendly interface

## Technologies Used
- HTML5
- SCSS (Sass)
- JavaScript (Vanilla JS)
- Vite (Development build tool)
